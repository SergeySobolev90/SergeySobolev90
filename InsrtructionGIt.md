## Слияние веток
Git позволяет нам сливать ветки между собой.
Есть главная ветка. От нее в какой-то момент создают второстепенную, в которой происходят изменения. Как только работа сделана, нужно слить одну ветку в другую.
Для этого используется команда:
> git merge 

Так же мы должны указать название ветки, которую мы хотим слить с текушей веткой.

## Разрешение конфликта 
При работе с Git могут возникать конфликты при слиянии двух веток, это происходит потому что в обоих ветках может присутсвовать информация и при слиянии если Git самостотельно не может разрешить конфилкт, он предлагет нам принять окончательное решение и предоставляет нам выбор
